# 🏪 클리어런스 매장 관리 시스템

실시간 재고 현황 및 판매 분석을 위한 웹 대시보드입니다.

## 🚀 주요 기능

### 📊 대시보드 기능
- **실시간 통계**: 총 상품 수, 매출액, 평균 할인율, 총 판매량
- **매장별 분석**: 매장별 매출 현황 비교
- **카테고리 분석**: 상품 카테고리별 판매량 분포
- **할인율 분석**: 할인율 분포 히스토그램
- **판매 추이**: 기간별 평균 판매량 추이

### 🔍 필터링 기능
- **매장별 필터**: 나이키일산, 브이엠플라스연수, 모다아울렛대전
- **BIZ별 필터**: AP(아페릴), FW(풋웨어), EQ(이퀄먼트)
- **카테고리별 필터**: 신발, 운동화, 라이프스타일, 농구화, 캐주얼
- **상품명 검색**: 실시간 텍스트 검색

### 📁 데이터 관리
- **CSV 파일 업로드**: 20개 컬럼 구조 지원
- **샘플 데이터**: 테스트용 샘플 데이터 제공
- **데이터 다운로드**: 필터된 결과 CSV 다운로드
