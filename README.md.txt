# 🎯 재고적정성 관리 시스템

8단계 분류 기반 스마트 재고 관리 대시보드

## 🌟 주요 기능

- ✅ **실시간 재고적정성 8단계 분석**: A급(소진대상) ~ H급(폭탄) 자동 분류
- ✅ **BIZ별 맞춤 할인 전략**: AP/FW/EQ별 차별화된 할인율 제공
- ✅ **스마트 계산기**: 할인율/수익률/마진 실시간 계산
- ✅ **자동 알림 시스템**: 과다재고/품절위험 자동 감지
- ✅ **FOS 엑셀 연동**: 실제 데이터 업로드 및 자동 분석
- ✅ **채널별 최적 배분**: FOS/대전모다/LF/온라인 운영 가이드

## 🔐 접속 정보

- **웹사이트**: [inventory-adequacy.streamlit.app](https://inventory-adequacy.streamlit.app)
- **비밀번호**: `dy1234`
- **세션 유지**: 브라우저 세션 기반

## 📊 8단계 재고적정성 분류

| 등급 | 재고량 | 점수 | 상태 | 관리전략 |
|------|--------|------|------|----------|
| A급 | 50개 미만 | 70점 | 소진대상 | 신속정리 |
| B급 | 50-99개 | 75점 | 정리예정 | 계획적 소진 |
| C급 | 100-199개 | 85점 | 적정 | 표준 운영 |
| D급 | 200-499개 | 90점 | 안정 | 안정적 관리 |
| E급 | 500-999개 | 95점 | 최적 | 적극 확대 |
| F급 | 1000-1999개 | 80점 | 많음 | 적극 판매 |
| G급 | 2000-3999개 | 50점 | 과다 | 할인 검토 |
| H급 | 4000개 이상 | 20점 | 폭탄 | 긴급 처분 |

## 💰 BIZ별 최대 할인율

### AP (Apparel)
- BETTER 신상품: 20-25% → 최대 50%
- 부진 신상품: 30-40% → 최대 53%
- 이월 상품: 45-55% → 최대 60%

### FW (Footwear)  
- BETTER 신상품: 25-30% → 최대 42%
- 부진 신상품: 35-45% → 최대 49%
- 이월 상품: 50-60% → 최대 62%

### EQ (Equipment)
- BETTER 신상품: 10-15% → 최대 32%
- 부진 신상품: 20-30% → 최대 47%
- 이월 상품: 35-45% → 최대 51%

## 🏪 채널별 운영 전략

| 채널 | 운영 대상 | 할인율 | 특징 |
|------|-----------|--------|------|
| **FOS** | A-C급 | 정가~30% | 브랜드 가치 보호 |
| **대전모다** | B-C급 | 30-60% | 아웃렛 할인 판매 |
| **LF** | C급 | 35-55% | 온라인 특가 |
| **온라인** | D급 이상 | 50%+ | 대량 재고 처분 |

## 🚀 사용 방법

1. **접속**: https://inventory-adequacy.streamlit.app
2. **로그인**: 비밀번호 `dy1234` 입력
3. **데이터 업로드**: FOS 엑셀 파일 업로드 (선택사항)
4. **필터 설정**: BIZ/등급별 필터 적용
5. **분석 확인**: 차트 및 테이블에서 현황 파악
6. **계산기 활용**: 할인율/수익률 실시간 계산
7. **리포트 다운로드**: CSV 형태로 데이터 내보내기

## 📧 알림 기능

- **실시간 알림**: 과다재고(F급 이상) 자동 감지
- **이메일 리포트**: 일일/주간 요약 리포트 발송
