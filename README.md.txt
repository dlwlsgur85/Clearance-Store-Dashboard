# 🏪 클리어런스 매장 관리 시스템

실시간 재고 현황 및 판매 분석을 위한 종합 웹 대시보드입니다.

## 🚀 8개 메뉴 구성

### 📈 1. 종합 대시보드
- **실시간 통계**: 총 상품 수, 매출액, 평균 할인율, 총 판매량
- **매장별 매출 현황**: 인터랙티브 바 차트
- **카테고리별 분포**: 파이 차트로 시각화

### 🏪 2. 매장별 실적
- **매장 선택**: 드롭다운으로 특정 매장 선택
- **매장 주요 지표**: 해당 매장의 상품 수, 매출액, 할인율, 판매량
- **카테고리별 성과**: 스캐터 플롯으로 성과 분석
- **일별 판매 추이**: 라인 차트로 트렌드 확인
- **상품 현황 테이블**: 매장별 상세 상품 목록

### 📊 3. 실적 분석
- **매출 트렌드 분석**: BIZ별, 성별 매출 비교
- **할인 효과 분석**: 할인율과 판매량 상관관계
- **재고 회전율 분석**: 효율성 측정
- **상품 성과 분석**: 다각도 성과 평가

### 📧 4. 이메일 리포트
- **자동 리포트 생성**: 주요 지표 요약
- **매장별 성과 요약**: 간결한 성과 정리
- **주요 알림 포함**: 과다재고, 판매부진 등
- **이메일 발송 기능**: 실시간 리포트 공유

### 📥 5. 스마트 분석
- **자동 분석 실행**: 원클릭 종합 분석
- **점수 자동 계산**: 상품별 종합 점수 산출
- **위험도 평가**: 재고 위험도 자동 분류
- **추천 액션**: AI 기반 액션 플랜 제시
- **분석 결과 다운로드**: CSV 파일로 결과 저장

### 🎯 6. KPI 점수
- **종합 점수 대시보드**: 시각적 점수 카드
- **5개 핵심 지표**: 종합점수, 재고회전율, 할인효율성, 매출성과, 재고리스크
- **레이더 차트**: 다차원 성과 시각화
- **개선 제안**: 자동 개선점 제시

### ⚠️ 7. 알림 센터
- **자동 알림 감지**: 과다재고, 품절위험, 고수익 기회
- **알림 분류**: 위험도별 색상 구분 (빨강/노랑/초록)
- **상세 항목 표시**: 해당 상품 목록 제공
- **알림 설정**: 사용자 정의 임계값 설정

### 🧮 8. 스마트 계산기
- **할인율 계산기**: 원가/판매가 기반 할인율 계산
- **수익률 계산기**: 목표 수익률 달성 가격 계산
- **마진 계산기**: 총 마진, 마진율, ROI 계산
- **재고회전율 계산기**: 재고 효율성 및 개선 방안

## 🎯 핵심 기능

### 📊 자동화된 분석
- **실시간 KPI 계산**: 5개 핵심 지표 자동 산출
- **위험도 자동 평가**: 재고 리스크 자동 분류
- **성과 점수화**: 상품별 종합 점수 계산

### ⚠️ 스마트 알림 시스템
- **과다재고 감지**: 180일 이상 재고 자동 감지
- **판매부진 경고**: 일일 판매량 0인 상품 알림
- **고수익 기회**: 추가 할인 여지 상품 식별

### 📧 자동 리포팅
- **일일 요약 리포트**: 주요 지표 자동 요약
- **매장별 성과**: 매장별 실적 정리
- **이메일 발송**: 자동 리포트 이메일 전송

### 🧮 실시간 계산기
- **할인 최적화**: 목표 할인율 달성 가격 계산
- **수익성 분석**: ROI 및 마진 실시간 계산
- **재고 효율성**: 회전율 개선 방안 제시
